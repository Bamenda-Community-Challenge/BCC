Team EPSILON

Names of Membres

Epsilon a custom made laser engraver. With it, we are able to create PCBs (printed Circuit Boards), Artistic drawings and carvings on wood, leather, plastic, glass, jeans and even on metal. It is controlled using an Arduino to send signals to stripper motors and also a laser diode. The machine consist of two axis (the x and y axis) and with the help of the laser, it can produce 2D engravings.

SWIMS

Names of Membres

SWIMS is taking an action by leveraging the Internet of Things (IoT), Robotics, and Artificial intelligence (AI) to build a flexible and adaptive monitoring system to improve waste collection in any environment. This system uses sensors, for example, waste-level monitoring and waste weight measuring in smart dustbins, to monitor data from disposal sites. By leveraging the Internet of Things (IoT), we collect this data in a waste planner and forecasting software where the data is processed. Furthermore, by leveraging machine learning, the system produces smart and lower-cost collection plans using the collected data. Another innovative feature is waste forecasting. this feature deploys an agent using machine learning to give future projections about collections sites on the system. For example, with waste forecasting, we can know the latest dustbins that will be full in the nearest hour and the collection plan to use.
